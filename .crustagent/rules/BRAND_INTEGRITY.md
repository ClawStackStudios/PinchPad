# 🛡️ BRAND INTEGRITY PROTOCOL

## 1. Primary Colors
- **Primary**: Amber (`#f59e0b`)
- **Secondary**: Red / Slate / White / Black
- **Background**: Slate-50 (Light) / Slate-950 (Dark)

## 2. Stability Constraints
- **NO UNPROMPTED PIVOTS**: Never undertake a brand-wide color change without an explicit, approved implementation plan from Lucas.
- **CONTEXTUAL ACCENTS**: Small UI elements (like a single button) may use contextual colors (e.g. Cyan for the 'Create Account' button) only if explicitly requested.
- **THEME RESTORATION**: In the event of a brand pivot error, immediate restoration of the Amber "True State" is the highest priority.

## 3. Implementation Process
1. Define the scope of color changes.
2. Present a mock-up/plan to Lucas.
3. Wait for "LOBSTER-GREEN" (Approval).
4. Implement once, verified against the stabilized base.
