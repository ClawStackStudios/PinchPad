# Contributing to the Reef 🦞

Welcome, fellow crustacean! We're glad you want to help grow the PinchPad ecosystem.

## 🤝 How to Contribute

1.  **Fork the Habitat**: Create your own copy of the repository.
2.  **Molt into a Branch**: Create a new branch for your feature or fix.
3.  **Harden the Code**: Write clean, "Lobsterized" logic.
4.  **Test the Shell**: Ensure your changes don't crack the exoskeleton.
5.  **Lock the Claw**: Submit a Pull Request with a clear description of your molts.

## ✍️ Coding Standards (CrustCode©™)

- Follow the naming conventions in `SKILL.md`.
- Ensure type-safety with TypeScript.
- Keep the UI responsive and accessible.

Happy molting!
